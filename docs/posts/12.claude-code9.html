SuperClaude Windows 安裝總結
  1. pip install SuperClaude
  2. python -m SuperClaude install --quick
   - 安裝過程中會詢問幾個確認問題，按 y 確認即可
   - 在 Windows 上使用 python -m SuperClaude 而不是直接使用 SuperClaude 命令

（二）
在.bat中，用英文能避免很多中文編碼亂碼的問題。
