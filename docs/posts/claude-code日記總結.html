使用心得總結



1. 優點：
- 需求分析和總結能力很强
- Git理解能力厲害
- 代碼分析功能強大
- 對配置文件理解深刻

2. 注意事項：
- 對掌握不到的數據記憶不清晰，需要統一命名規範
- 使用Context7容易超出Token限制
- 最好先思考5分鐘再下指令，效果更好
- 必須開啟Git管理，避免代碼丟失

3. 價值評估：
- 每月20美元訂閱費用
- 實際使用價值約60美元/週
- 年費相當於每天120-130日元（一杯咖啡的價格）
