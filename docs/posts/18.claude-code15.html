關於Claude Code | Claude Pro 第十五天

2025-08-16 21:12:18
想做一個小兔几做月餅的月餅工廠的小游戲。用Kiro試了一下，通過下一步下一步做出來了，不像ClaudeCode做的那樣簡陋的，很多功能，游戲引擎都給編寫出來了。
但是最重要的美工還是差强人意，角色的插圖和背景仍需要自己創作。

用Claude的網頁版做了一個在Windows系統中的AI協助冥想系統.exe的演示用的HTML。功能組件完善花了一整天時間 前前後後改了40幾個版本，API次數限制也一再提醒。
最終仍卡在一些細小的部件顯示錯誤上，很可惜，要明天才能繼續使用Claude4.1來繼續修改了。不過完成度很高，很喜歡。
